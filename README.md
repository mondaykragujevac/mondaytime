# mondaytime

Monday — auto škola Kragujevac.

## Stack
- HTML5 statički sajt
- Tailwind CSS v4
- DaisyUI 5
- Vercel hosting

## Build
```bash
npm install
npm run build     # jednokratni build
npm run dev       # watch mode
```

Build output: `dist/output.css`.

## Deploy
Auto-deploy na Vercel pri svakom `git push` na `main`.
